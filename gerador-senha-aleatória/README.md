# Password Generator

Gerador de senha aleatória simples, desenvolvido com HTML, CSS e JavaScript.

## Funcionalidades

- Gera senhas aleatórias
- Copia a senha para a área de transferência
- Interface simples e direta

## Como usar

1. Abra o arquivo index.html no navegador.
2. Clique no botão para gerar uma senha.
3. Use o ícone de cópia para copiar o resultado.

## Melhorias planejadas

- Escolha do número de caracteres
- Seleção de tipos de caracteres
- Opção para gerar senhas mais seguras

## Arquivos

- index.html: estrutura da interface
- style.css: estilos da página
- app.js: lógica de geração e cópia da senha
